Learn

Learn Kotlin with small simple annotated examples. Click on the left navigation menu to go to a specific sample. Make sure you check back often, as the website gets updated frequently.

Contribute

Source code on GitHub

1. Create a new markdown file. 2. Give it a title. 3. Create the [sample](https://github.com/kotlinbyexample/kotlinbyexample.github.io/blob/master/_examples/classes.md). 4. Add the entry to the [Examples index](https://github.com/kotlinbyexample/kotlinbyexample.github.io/blob/master/_data/examples.yml) 5. Send a Pull Request!
